Sandbox Application

Network Configuration Instructions

Configure the network for your sandbox application as described below:

  • Enable router configuration commands for fastethernet and serial interfaces.
  • Set static IP addresses for all devices.
  • Verify connectivity using ping commands.

IP Address Table

Device Role IP Address Subnet Mask
Desktop VM Management 192.168.40.2 255.255.255.0
Gateway Router (enp0s3) Internet Access 10.0.0.16 255.255.255.0
Gateway Router (enp0s8) Subnet 01 192.168.40.1 255.255.255.0
Gateway Router (enp0s9) Subnet 02 192.168.140.1 255.255.255.0
Application Server Server 192.168.140.2 255.255.255.0

Testing and Verification

Ensure connectivity by performing ping tests between devices:

  • Ping 192.168.40.1 from Desktop VM.
  • Ping 192.168.140.2 from Ubuntu Server.
  • Ping 192.168.40.2 from Application Server.
